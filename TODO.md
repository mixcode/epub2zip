
## Overview

Make a command-line Go utility program that reads a epub file, check whether it is a fixed-layout book that each page is composed of single image, like illustrations or comic books. If it is a fixed page book, extract each page image, set the filename of the image as the page number, and store all pages in a new ZIP file.



## Misc

* EPUB 3.3 Specification: https://www.w3.org/TR/epub-33/

* See `/example_epub` directory for samples
